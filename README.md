# kingbot
Jangan lupa subscribe channel channel saya
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/KingIdBot/kingbot
> cd kingbot
> npm install
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```

---------

### FOR WINDOWS/VPS/RDP USER
```bash
> git clone https://github.com/KingIdBot/kingbot
> cd kingbot
> npm install
```
###### Run
```bash
> node index.js
```
